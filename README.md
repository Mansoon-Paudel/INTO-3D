# Head Hunter
 
An 3D game where you find dragon skulls,
 
**How it works:** FInd the heads scattered across the map and done!
 
---
 
## Controls
- Move:  W / S / A / D <br>
- Jump:  Space <br>
- Sprint: Shift+Move
- Freecam: Colon(:)
 
---
 
## Features
- 3D enviroment
- Freecam and Sprint
---
 
*Built with Godot 4 & GDScript.*
